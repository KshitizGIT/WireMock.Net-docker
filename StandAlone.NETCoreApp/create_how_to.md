# Nano
## Commands
### 1. Build
```
docker build -t sheyenrath/wiremock.net-nano  -f .\Dockerfile.nano .
```

### 2. Tag
```
docker tag sheyenrath/wiremock.net-nano:latest sheyenrath/wiremock.net-nano:1.0.4.13
```

### 3. Push
```
docker push sheyenrath/wiremock.net-nano:1.0.4.13
```
